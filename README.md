# Energy Futures Calculator

Rust prototype for energy futures valuation, portfolio risk and commodity tokenization research aligned with Stellar infrastructure.

## Scope

- Futures valuation
- PnL simulation
- Risk engine prototypes
- Path toward tokenized commodity settlement on Stellar

## Roadmap

### Phase 1
- Contract notional calculator
- PnL engine
- Portfolio exposure model

### Phase 2
- Risk metrics
- Liquidity pool simulation
- Tokenized settlement concepts

## Long-term Vision

Bridge traditional energy markets and on-chain financial systems using Rust and later Soroban/Stellar.
